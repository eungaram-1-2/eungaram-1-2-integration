개인정보처리방침

시행일: 2026년 4월 25일  |  은가람 중학교 1학년 2반

제1조  개요

은가람 중학교 1학년 2반(이하 "우리 반")은 학급 통합 사이트(이하 "사이트")를 운영하면서 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」을 준수합니다. 본 방침은 사이트에서 수집하는 정보의 종류, 이용 목적, 보호 방법 등을 안내합니다.

제2조  수집하는 정보

사이트는 다음과 같은 정보를 수집하거나 처리할 수 있습니다.

  • 자동 수집 정보 (필수)
    • 접속 로그 (IP 주소, 접속 일시, 사용 브라우저/기기 정보)
    • 페이지 방문 기록, 오류 로그
  • 사용자 작성 콘텐츠 (선택적)
    • 게시판 글 제목 및 내용, 댓글
    • 채팅 메시지 (AI 채팅 기능 이용 시)
    • 투표 참여 기록
  • 로컬 저장 정보 (기기 내에만 저장)
    • 사용자 설정값 (다크모드, 알림 여부 등) - 브라우저 로컬스토리지 저장
    • CSRF 보안 토큰 - 세션 종료 시 자동 삭제

사이트는 별도의 회원가입 절차가 없으며, 이름·연락처·주민등록번호 등 민감한 개인식별정보를 수집하지 않습니다.

제3조  정보 수집 목적 및 이용

  • 급식 정보, 시간표, 학사일정 등 학급 서비스 제공
  • 비정상적인 접속 및 어뷰징 방지 (IP 차단, Rate Limiting)
  • 서비스 오류 진단 및 품질 개선
  • 부적절한 콘텐츠 필터링 (욕설, 스팸 등)

수집된 정보는 위 목적 이외의 용도로 사용하거나 제3자에게 제공하지 않습니다.

제4조  Firebase 서비스 이용 안내

사이트는 게시판, 채팅, 투표 등 실시간 기능 제공을 위해 Google의 Firebase Realtime Database 및 Firebase Storage를 사용합니다.

  • Firebase는 Google LLC가 운영하는 클라우드 서비스입니다.
  • 게시판·채팅에 작성된 데이터는 Firebase 서버(미국 또는 아시아 리전)에 저장될 수 있습니다.
  • Firebase의 개인정보처리방침: https://firebase.google.com/support/privacy
  • Google의 개인정보처리방침: https://policies.google.com/privacy

제5조  쿠키 및 로컬스토리지

사이트는 쿠키(Cookie)를 사용하지 않습니다.

대신 브라우저의 로컬스토리지(LocalStorage)와 세션스토리지(SessionStorage)를 사용합니다.

  • LocalStorage: 사용자 환경 설정(테마, 알림 등)을 브라우저에 저장합니다. 직접 삭제하기 전까지 유지됩니다.
  • SessionStorage: CSRF 보안 토큰 등 세션 중 임시 데이터를 저장합니다. 브라우저 탭을 닫으면 자동 삭제됩니다.

로컬스토리지 데이터는 사용자 기기 안에만 저장되며, 서버로 전송되지 않습니다.

제6조  정보 보유 기간

  • 접속 로그: 최대 3개월 보관 후 자동 삭제
  • 게시글·댓글·채팅 메시지: 작성자 요청 또는 관리자 판단 시 즉시 삭제
  • 로컬스토리지 데이터: 사용자가 직접 삭제하거나 브라우저 데이터를 초기화할 때까지 유지

제7조  이용자의 권리

사이트 이용자는 다음 권리를 행사할 수 있습니다.

  • 본인이 작성한 게시글·댓글의 열람, 수정, 삭제 요청
  • 개인정보 처리와 관련한 이의 제기
  • 로컬스토리지 데이터는 브라우저 설정에서 직접 삭제 가능

제8조  보안 조치

사이트는 개인정보 보호를 위해 다음 기술적 보안 조치를 적용합니다.

  • HTTPS(TLS) 암호화 전송 (HSTS 적용)
  • XSS(크로스 사이트 스크립팅) 방지 - 입력값 검증 및 HTML 이스케이프 처리
  • CSRF 토큰을 통한 요청 위변조 방지
  • Rate Limiting - 과도한 요청 자동 차단
  • 부적절한 IP 자동 차단
  • Content-Security-Policy(CSP), X-Frame-Options 등 보안 헤더 적용

제9조  외부 서비스 링크

사이트에는 NEIS(교육정보시스템), Google Sheets 등 외부 서비스 링크가 포함될 수 있습니다. 외부 사이트로 이동한 이후의 개인정보 처리에 대해서는 해당 사이트의 방침이 적용되며, 우리 반은 책임을 지지 않습니다.

제10조  방침 변경

본 방침은 법령·서비스 변경에 따라 개정될 수 있습니다. 변경 시 사이트 공지사항을 통해 사전 안내하며, 시행일부터 효력이 발생합니다.
